# 하이록스 더블 대비 8주 러닝 & 크로스 트레이닝 프로그램

## 목표
- 10km 45분 이내 달성 (1km 4:30 페이스)
- 하체 근지구력 및 인터벌 능력 강화
- 러닝 후 피로 상태에서의 하체 운동 적응

---

## 주차별 세부 프로그램

### 표기 설명
- **인터벌**: 1km 전력(주차별 페이스), 세트간 2~3분 걷기/조깅
- **지속주**: 편안한 페이스로 일정 거리 달리기
- **페이스런**: 목표 페이스로 10km 달리기(처음엔 5km부터 시작 가능)
- **크로스 트레이닝**: 자전거, 수영, 로잉 등(40분 내외)
- **하체 근지구력**: 런지, 스쿼트, 월싯 등(세트 수는 주차별 표기)

## 참고/팁
- 각 주차별 페이스가 너무 힘들면, 1~2주 반복 후 다음 단계로 넘어가도 됩니다.
- 페이스런은 처음엔 5km로 시작해 점차 10km까지 늘려도 무방합니다.
- 부상 방지 위해 스트레칭, 회복 신경쓰기.
- 크로스 트레이닝은 자전거, 수영, 로잉 등으로 다양하게 진행하세요.
